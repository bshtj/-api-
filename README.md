## 基于高德地图开发平台的校园地图web端开发

本项目针对新生和来校参观的游客。

具体实现功能还在商讨中。

--------------

#### 1、地图

基础的地图图层，加上3D转换。包含楼宇、道路、车站等信息。

#### 2、点击事件

- 显示地点信息（固定在侧边框）
- 收藏地点（尝试与课表结合现实上课楼宇）
- 地点信息选择性显示（通过勾选选择性显示）
- 选中地点高亮显示
- 导航前往选中地点

#### 3、课表

- 导入课表方式（暂时搁置）
- 搜索或点击显示相应时段上课地点，导航前往

#### 4、侧边栏

- 显示点击地点信息
- 包含搜索框，可以搜索热门问题（如校内快递点）
- 显示其他信息如天气等
